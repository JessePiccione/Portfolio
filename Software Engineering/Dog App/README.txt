Project description:
	This application is a Dog information Application using axios and cheerio to scrape content off of wikipedia. It may need adjustments to work depending on how wikipedia has its pages is set up now versus when this was made 2021ish.
Run:
To execute this program you need to have node js installed on your machine then navigate to the cwd of the projects app.js file and run the following commands

1. npm install
2. npm start
