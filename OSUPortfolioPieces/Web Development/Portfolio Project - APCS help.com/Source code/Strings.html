What's a String?

A String is an array of chararaters with a quiet large data limit around 65535 bytes constructed from concantination in prgramming languages. One the best ways to deal with a string is to treat it as array of chars or other things. Treat str.charAt(i) like a charArray[i] but with the proper method. To concatinate a string to some other thing:

               
//take a string like
String str = "name: ";
//add Some Data 
str += "John Doe";

After those two lines complete the value of str is what?


ANSWER: "name: John Doe"

Concatatination is the action of combining two strings. This makes strings highly useful as opposed to a Char[] because we do not have to look at each char as thier own index. With Strings each char has a index but printing is treated as one object. However to declare a String s can be declared with new String("Hello World");. Strings have many different methods built into the class. Here are some:

String Methods

  • String substring(int start, int end) - returns a part of a string from start to end indexes not including
  • String substring(int start) - returns a part of a string from start to end of string
  • int length() - returns length in characters of this
  • int indexOf(String str) - returns index of first occurance of str from this
  • int lastIndexOf(String str) - returns index of last occurance of str from this
  • int compareTo(String str) - returns 0 if the objects are equal, < 0 if this is less than str, > 0 if this is greater then str
  • boolean equals(String str) - returns true if strings are equal, otherwise false
  • String toLowerCase() - returns a String with are characters now lower case
  • String toUpperCase() - returns a String with all upper case charaters
  • boolean contains(String str) - returns true if this has substring of str
  • int charAt(int i) - returns character located in index i
